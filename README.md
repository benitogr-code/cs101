## Computer Science 101

This project implements a small collection of data structures and typical computer science algorithms / problems.

### Data Structures

* Fix Array
* Dynamic Array
* Linked List
* Stack
* Queue
* Hash Map
* Binary Search Tree (BST)
* AVL Tree
* Min Heap
* Trie
* String Builder
* Big Integer

### Algorithms / Problems

* Sorting
	* Select
	* Bubble
	* Insert
	* Merge
	* Quicksort
	* Heap sort
* Eight Queens
* Knapsack
* Misc problems: Strings, Bit manipulation, ...
